Aide mémoire des selecteurs CSS déjà vus

  • * selectionne tout les éléments de la page
  • .ma-jolie-nav sélectionne tous les éléments ayant le nom de class "ma-jolie-nav"
  • #mon-unique-id délectionne tout les éléments ayant le nom d'id "mon-unique-id"
  • div, article, aside sélectionne toutes les balises <div>, <article> ET <aside>
  • nav a sélectionne toute les liens <a> inclu dans la balise <nav>
  • nav a hover sélectionne tout les liens dans la nav et active une animation lors du survol der la souris
